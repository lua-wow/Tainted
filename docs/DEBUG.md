# Debugging

-   [Interface action failed because of an AddOn](https://www.bluetracker.gg/wow/topic/us-en/2518903044-interface-action-failed-because-of-an-addon/)

## Enable TaintLog

```txt
/console taintLog 1
```

> search through the file `Logs\taint.log`  for the word `blocked`

## Disable TaintLog
```txt
/console taintLog 0
```
